# Claude Code Instructions

This file is read by Claude Code on session start.

**The single source of truth for this project is `AGENTS.md` in the repository root.**

Before suggesting any code change, architectural choice, dependency, or convention, read `AGENTS.md` and follow it. If a request from the user appears to conflict with `AGENTS.md`, flag the conflict and propose either an ADR under `docs/decisions/` or alignment with the existing rule.

Specs for in-progress work live under `docs/specs/`. The current weekend's spec is the contract for what to build; do not exceed its scope.

Architecture decisions live under `docs/decisions/` (ADRs). Treat them as binding unless explicitly superseded.
